HTML & CSS - A Quick Reference Guide

This is a Reference guide for HTML & CSS. This guide would be helpful in case you have forgotten any of the commands or you haven't done HTML & CSS in a while, it will be really useful during those types for you to recap. This guide is based of the knowledge I have so far in HTML & CSS.

HTML Tags

1. Heading - <h1> to <h6>

The heading tag are of 6 types, each of them comes in different sizes. It is from <h1> to <h6>.

Code:

     
        <h1>Heading1</h1>
        <h2>Heading2</h2>  
        <h3>Heading3</h3>
        <h4>Heading4</h4>
        <h5>Heading5</h5>
        <h6>Heading6</h6>

Result:

Heading1

Heading2

Heading3

Heading4

Heading5
Heading6

We can use different heading sizes according to the requirement.
The Heading Tag ends with h6 after that it will be shown in normal text.

2. Paragraph - <p>

The <p> is used to write paragraphs. It basically helps in separating the writing into paragraphs

Code:


<p> Roses are red, Violets are blue </p>

Result:

Roses are red, Violets are blue

The speciality of<p> tag is that, even if it is written in next line, it will show in one line even if there is extra space more than the default space it will be omitted.

Code:


    <p>
        Roses are                      red,
        Violets are blue
    </p>

Result:

Roses are red, Violets are blue

3. Italics - <i>

The <i> tag is used to make a paragraph in italics.

Code:

<i> Lorem ipsum dolor sit amet consectetur adipisicing elit. 
    Rerum fugiat voluptate eum impedit consequuntur provident vitae, neque nulla. 
    Quidem inventore et illo earum sequi perferendis corporis excepturi perspiciatis quod voluptates! </i>

Result:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Rerum fugiat voluptate eum impedit consequuntur provident vitae, neque nulla. Quidem inventore et illo earum sequi perferendis corporis excepturi perspiciatis quod voluptates!

4. Bold - <b>

The <b> tag is used to give more bolder look to the text.

Code:

<p>I like <b>Roses.</b></p>

Result:

I like Roses.

Here the word "Roses" is seen in bold text with the help of <b> tag.

5. Ordered List - <ol>

The <ol> tag is used to make the list in order and it can be ordered alphabetical or numerical. With this tag we use the <li> which is called as list to define each item.

Code:


<ol>
    <li>Pencil</li>
    <li>Ben</li>
    <li>Book</li>  
</ol>

Result:

  1. Pencil
  2. Pen
  3. Book

6. Unordered List - <ul>

The <ul> tag is used to make an unordered list. It is mostly shown in bullets. We use the <li> tag here as well since it is a list.

Code:


    <ul>
        <li>Cake</li>
        <li>Ice Cream</li>
        <li>Chocolate</li>   
    </ul>

Result:

  • Cake
  • Ice Cream
  • Chocolate

7. Image - <img>

The <img> tag is used to show images. We can insert images in the webpage.
The img tag has two attributes:

  1. src: It shows the source/link of the image.
  2. alt: It is the alternate text for the image, in case if the image cannot be seen by the viewer.

Code:


<img src="https://images.unsplash.com/photo-1741527383649-b0d7c54d12ef?q=80&w=1974&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8
fA%3D%3D" alt="Group of flowers" height="400" width="300">

Here the height and width is used to define the size of the image.

Result:

8. Anchor - <a>

Anchors are defined using <a> tag with href attribute which shows the link.

Code:

<a href="https://www.google.com">Google</a>

Result:

Google

9. Line Break - <br>

The <br> tag is used for line breaks. It shows the text in the next line.

Code:


<p>Roses are red, <br>
Violets are blue</p>

Result:

Roses are red,
Violets are blue

10. Button - <button>

The <button> tag is used as a button that you can click.

Code:

<button type="button">Print</button>

Result:

CSS Properties

1. color

Using the color property, we can set color to the text.

Code:


            <p style="color:blue">Creating Websites is Fun!</p>

Result:

Creating Websites is Fun!

2. background-color

The background-color is used to give color to the background of the page, section, division and many more.

Code:


            body {
                background-color: aquamarine;
            }

Result:

Note: When you do it in CSS, it will cover the whole page, this is just an example.

3. border

This border property is used to give borders for a section, div or depending to the requirement.

Code:


            p {
                border:solid green;
            }

Result:

4. font-family

font-family helps to change the default font to the font we like.

Code:


            body{
                font-family: Lucida Sans', 'Lucida Sans Regular', 'Lucida Grande', 'Lucida Sans Unicode', Geneva, Verdana, sans-serif;;
            }

Result:

This font is different from the default font.

5. text-align

text-align helps the code to be aligned where the user would prefer to place it.

Code:


            p{
                text-align: center;
            }

Result:

This text is in the center.

6. margin

The margin property is used for the spacing of the white space around the element.

Code:


            p{
                margin: 50px;
            }

Result:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolores facilis mollitia laudantium error corporis nulla exercitationem, dolore incidunt temporibus, odit accusamus eaque amet provident. Praesentium unde expedita architecto molestias ipsum.

It gives equal spacing in all 4 sides.

There is also other properties, in case you only want to give spacing to one side or more or use different pixels for each side, you can use:

  • margin-top
  • margin-bottom
  • margin-right
  • margin-left

7. filter (Image Filters)

The filter property is used to give filter to the images.

Code:


            img{
                filter: grayscale(2.0);
            }

Result:

Note: You can also try it, with other filter such as:

  • blur()
  • brightness()
  • grayscale()
  • sepia()
  • drop-shadow()
  • hue-rotate()
  • opacity()
  • invert()
  • saturate()
  • contrast()

8. font-size

The font-size property is used to set the size of the text.

Code:


            p{
                font-size: large;
            }

Result:

You can see the size of the text is bigger than the default text size.
